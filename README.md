📚 Formulário de Lançamento de Notas (HTML)
Objetivo:
Criar uma interface simples e funcional que permita inserir e visualizar informações acadêmicas, seguindo regras específicas de validação e usabilidade.

🛠️ Funcionalidades
Seleção de disciplina por meio de uma lista pré-definida:
Introdução à Informática
Linguagens de Programação
Programação de Computadores

Campos para inserção de notas:
- Prova 1
- Prova 2
- Exame
Aceitam apenas valores numéricos entre 0 e 10

Campo de média:
- Calculado automaticamente
- Não permite edição pelo usuário (readonly)

Campo de faltas:
- Aceita apenas números entre 0 e 16

Validações:
- Campos obrigatórios: Prova 1, Prova 2 e Faltas
- Restrição de valores com uso de atributos HTML (min, max, required)

💡 Tecnologias utilizadas
HTML5

📌 Observações
O projeto utiliza recursos nativos do HTML para validação de dados, sem necessidade de JavaScript, focando em simplicidade e boas práticas na construção de formulários.
